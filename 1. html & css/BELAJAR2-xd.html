3. list html

Dalam html, list merujuk pada elemen yg digunakan untuk menyusun item item dalam format yg tersruktur. ada beberapa jenis list yang bisa digunakan

1. ordered list

ordered list yaitu list yg dibuat secara berurutan dibuat dengan tag ol lalu di dialam nya berisi dengan item yang dimasukkan yg dibuat dengan menggunakan tag li, berikut, contoh penggunaan tag ol.

ordered List
  1. list 1
  2. list 2
  3. list 3

  1. list 1

  3. list 3

  1. list 1
  2. list 2
  3. list 3
ini ordered list tapi pake romawi besar
  1. list 1
  2. list 2
  3. list 3
ini ordered list tapi pake romawi kecil
  1. list 1
  2. list 2
  3. list 3
